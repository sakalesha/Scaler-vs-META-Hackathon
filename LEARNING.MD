# What is the purpose of models.py in this project?
# This file defines the data models for the clinical triage environment.
# It includes:
# - Enumerations for ESI levels and action types
# - Patient state model
# - Environment state model
# - Action model
# - Observation model
# - Task description model
# - Grader request and response models
# - Baseline request and response models   

# What is ESI in this context?
# ESI is a 5-level triage system used in emergency departments to prioritize patients based on the severity of their condition.
# 1 - Immediate
# 2 - Emergent
# 3 - Urgent
# 4 - Less urgent
# 5 - Non-urgent

# What is triage in this context?
# Triage is the process of determining the priority of patients' treatments based on the severity of their condition.

# What is baseline in this context?
# Baseline is a simple AI agent that can be used to compare against the trained RL agent.
# It is a rule-based agent that can be used to compare against the trained RL agent.
